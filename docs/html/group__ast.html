Enumeration Type Documentation

enum BrainfuckInstruction::BrainfuckType

This enum defines the instruction set that should be implemented by a proper implementation of this specification.

Enumerator
INC 

Increase cell value.

DEC 

Decrease cell value.

MOVL 

Move left in the memory.

MOVR 

Move right in the memory.

CLR 

Clear cell value.

IN 

Read one byte from a stream and write to cell.

OUT 

Read one byte from cell and write to a stream.

JMP 

Unconditional jump.

JZ 

Conditional jump: if zero.

NOP 

No operation.
